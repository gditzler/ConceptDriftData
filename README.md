ConceptDriftData
================

Generate synthetic data sets containing concept drift, or load one of two real-world concept drift benchmark data sets.


About
================
The Matlab code provided in this project was written for recent work that has resulted in a manuscript being accepted in the IEEE Transactions on Knowledge and Data Engineering. The software is licensed under the GNU GPLv3. 


Demonstration
================
Refer to `gendatTKDE.m` for how to use `ConceptDriftData.m`.
